# AWS VPC Peering & Secure Networking Project

## 📌 Project Overview

This project demonstrates secure AWS cloud networking using VPC Peering, Public & Private Subnets, and Bastion Host Architecture.

The infrastructure enables secure communication between isolated VPCs and private EC2 instances using SSH.

---

## 🚀 Features

* Public & Private Subnets
* Bastion Host Architecture
* VPC Peering Configuration
* EC2 Instance Communication
* Route Table Configuration
* Security Group Configuration
* Secure SSH Access
* Ping & SSH Connectivity Testing

---

## 🛠 AWS Services Used

* Amazon VPC
* Amazon EC2
* VPC Peering
* Route Tables
* Internet Gateway
* Security Groups

---

## 🔐 Architecture Flow

User Laptop
↓
Public EC2 (Bastion Host)
↓
Private EC2
↓
Cross-VPC Communication using VPC Peering
